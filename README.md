# raspi-LEDsample（先端情報学実習-峰野クラス向けサンプル）
## 概要
raspberry piでLチカをするプログラムです。起動するとLEDが一秒置きに3回点滅します。

## 使い方
1. raspberry piのGPIOにLEDを接続（繋ぎ方は配布した資料を参考にしてください）
1. プログラムをダウンロード
    ```
    $ git clone https://github.com/MinenoLab/raspi-LEDsample.git
    ```
1. 実行
    ```
    $ cd raspi-LEDsample
    $ python led.py
    ```
